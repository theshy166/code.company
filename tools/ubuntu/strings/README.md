# String Resource Conversion

If a `strings` directory exists under the `product` directory, you need to 
use the `trans_strings_res.sh` script to perform the conversion.


## run script in root like bwlow，操作命令分别为：
	./tools/ubuntu/strings/trans_strings_res.sh $PRODUCT_NAME
    示例:
		./tools/ubuntu/strings/trans_strings_res.sh kidcamera_nor
